> Un <abbr title="Application Programming Interface">API</abbr> <abbr title="Representational State Transfer">REST</abbr> es un origen de datos que se puede consumir a través de una URL.

Esta mini app recibe el número de control de un estudiante y asigna una API aleatoria para practicar.

- Si llegaste aquí por casualidad, te invito a que pruebes la app con tu nombre.

## Asignaturas

Aplica para las siguientes asignaturas:

- Programación Web
- Programación Web Avanzada
- Desarrollo de Aplicaciones Web
- Desarrollo de Aplicaciones Móviles
- Desarrollo de Aplicaciones Multiplataforma
